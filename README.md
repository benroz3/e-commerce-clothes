Run with Docker:
1. Create .env using the example
2. Open Docker Desktop
3. Open Terminal
4. Type: 'docker build -t [choose-container-name] .'
5. Type: 'docker run -dp 3000:3000 [choose-container-name]'

Run without Docker:
1. Create .env using the example
2. Open Terminal
3. Type: 'npm i'
4. Type:'npm run dev'
